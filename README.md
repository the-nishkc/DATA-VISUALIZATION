# 🛒 Supermarket Sales Analytics Dashboard

A complete data analytics project built using **Python** and **Tableau** to analyze supermarket sales data. The project includes data cleaning, exploratory data analysis, professional visualizations, KPI dashboards, and an interactive Tableau dashboard for business insights.

---

## 📌 Project Overview

This project analyzes supermarket sales transactions to identify trends in revenue, customer behavior, payment methods, product performance, and branch-wise sales.

The analysis is performed using:

- Python
- Pandas
- Matplotlib
- Seaborn
- Plotly
- Tableau Public

## 🚀 How to Run

### 1. Clone the repository

```bash
git clone https://github.com/yourusername/Supermarket-Sales-Analytics.git
```

### 2. Install dependencies

```bash
pip install pandas matplotlib seaborn plotly
```

### 3. Run Python visualizations

```bash
python matplotlibvisualization.py
```

```bash
python seabornvisualizations.py
```

```bash
python plotlyvisualization.py
```

### 4. Open Tableau Dashboard

Open:

```
FINALDASHBOARD.twb
```

using Tableau Public/Desktop.
